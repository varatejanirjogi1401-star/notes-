# Day 1 - Python Basics

This folder covers the fundamentals of Python programming.

## Structure

- **`examples/`** — Basic syntax examples (print statements, variables)
- **`solved/`** — Solved exercises with explanations

- **`assignment/`** — Graded practice problems

- User input (`input()`)

- String formatting
